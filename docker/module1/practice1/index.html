What is Docker ?


Docker is a plateform that allows to build, test and deploy applications easily and fast..

For example if we build one application and it perfectly run in our machine but when we talk about other machine maybe it is not work perfectly for that docker is build so with the use of docker we can easily run applications in any machine .



What is container ?


Container is a heart of docker.

Containers are an abstraction at the application layer that pacages codes and dependenses together.

Multiple container can run on the same machine and share content with each otner containers .

The process is isolated in user space.



What are images ?


Docker images is a read only template that contain set of instructions for creating a container that run on thedocker plateform.

Docker Images contain application code , libraries , tools and other files needed to make an application run.



What is NGINX ?


It is an open source reverse proxy server for HTTP.

The nginx project started with a strong focus on high concurrency, high performance and low memory usage .



Commands:

Command Usage
docker attach Attach local standard input, output, and error streams to a running container
docker build Build an image from a Dockerfile
docker builder Manage builds
docker checkpoint Manage checkpoints
docker commit Create a new image from a container's changes
docker cp Copy files/folders between a container and the local filesystem
docker images List images
docker ps List containers
docker pull Download an image from a registry
docker run Create and run a new container from an image
